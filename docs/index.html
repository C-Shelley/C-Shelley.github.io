Projects



For my 4th Year Project I am creating a Book Tracker application that will allow users to search for books, preview them, add them to their library and track their progress. Users will be able to rate a book out of 5 stars and leave a review on it.

For my 3rd Year project I created a website using Django where people could buy and rent games. This website allowed users to search for a game, leave a comment on it and buy the game if they wanted. I utilised stripe in for purchasing a game. A user could rent a game for a period of time at a lower cost. I implemented some vouchers so users could get games at a reduced price.

For my 2nd Year project i created a blog website where people interested in art could come and talk about it and post their own art. This website was created using Django. It allowed users to search for a specific genre or mode of art like contemporary, and it would show all the art posted tagged with that. Users could like and comment on any art posts that they liked. Users could create an account which would allow them to comment not anonymously and customise their profile to their liking. A history of the users comments and likes would also be available if they had an account.
